3. Uploading Transaction FilesÂ¶

  • Click on InvestmentTransactionDownload
  • Click Add to add a new entry. You will be presented with the following menu.
  1. Choose a file to upload. This can be of type ofx,qfx or csv. For csv, a csv data definition is required.
  2. Choose an investment to link the transactions to.
  3. Choose a portfolio to link the transactions to.
  4. Start date and end date will be automatically set from the data file.

Note

This upload process is currently focussed on bank account and savings account data. Equity transactions are best entered manually.
